# deno_net

This crate implements networking APIs.

This crate depends on following extensions:

- "deno_web"
- "deno_fetch"

Following ops are provided:

- "op_net_accept"
- "op_net_connect"
- "op_net_listen"
- "op_dgram_recv"
- "op_dgram_send"
- "op_dns_resolve"
- "op_tls_start"
- "op_tls_connect"
- "op_tls_listen"
- "op_tls_accept"
- "op_tls_handshake"
